Reproduction Instructions

ethrex Benchmarks

  1. Clone ethrex-replay
  2. Run the prover:
cargo r -r -F "sp1,gpu" -p ethrex-replay -- blocks \
  --action prove \
  --zkvm sp1 \
  --from 23769082 \
  --to 23769092 \
  --rpc-url <RPC_WITH_DEBUG_EXECUTIONWITNESS>

RSP Benchmarks

  1. Clone rsp
  2. Run with CUDA:
SP1_PROVER=cuda cargo r -r \
  --manifest-path bin/host/Cargo.toml \
  --block-number <BLOCK> \
  --rpc-url <RPC> \
  --prove
